Overview

Get the last task of the stage template

Summary

This request will return the last task of the stage template

Request Arguments

Name Type Description Required
module String Record module True
record Object Object containing id of the current record True

Request Example

{
    "module": "DRI_SubWorkflow_Templates",
    "record": {
        "id": "e00ce644-2ce5-11ed-8ea1-0242ac120006"
    },
}

Response

The response will be an object containing the following fields:
Name Type Description
activityTemplate SugarBean Last DRI_Workflow_Task_Templates record bean

Response Example

{
    "activityTemplate" : {
        "id": "06e0898a-91f6-11e9-bff9-0ac615e576f2",
        "sort_order": "3",
        "points": "10",
        "url": "https://www.php.net",
    }
}
