Overview

[ADMIN] Upload a package from remote instance to local instance

Summary

Package builder uses this endpoint to upload a package from remote instance to local instance. This endpoint is only available to administrators.

Request Arguments

Name Type Description Required
connectionInfo array Remote instance's url, username, and password.
Example: {url:'http://localhost/sugarcrm',user:'abc',pass:'123'} 		True
stagedVersions array Staged versions of package. False
id string Package id. True

Response

Sample response if the package is uploaded successfully: 
    {
        "access": true,
        "fileInstallId": "56365210-4cce-11ea-a7bf-acde48001122"
    }
