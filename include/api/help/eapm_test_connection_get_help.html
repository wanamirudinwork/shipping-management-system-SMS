Overview

Checking the SMTP provider service connection is valid.

Summary

Endpoint is used to make a basic call to the service (getting Email Address).

Request

GET /EAPM/test?application=GoogleEmail&eapm_id=ba7931f0-dfe2-11ef-bf98-0242ac120

Request Arguments

Name Type Description Required
application String Type of application ('GoogleEmail', 'MicrosoftEmail') Yes
eapm_id String EAPM record ID Yes

Response

{
    "isConnected": false
}
