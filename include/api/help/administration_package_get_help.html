Overview

[ADMIN] Get a package

Summary

Use this endpoint to get the content of a package. This endpoint is only available to administrators.

Request Arguments

Name Type Description Required
id string Package id. True

Response

Package content is returned in the response as based64 encoded string. Sample response: 
{
    "name": "project_P1.zip",
    "content": "UEsDBAoAAAAAAFk0NlopkO0zlgkAAJYJAAAMAAAAbWFu"
}
