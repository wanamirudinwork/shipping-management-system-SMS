Overview

[ADMIN] Extract customizations

Summary

Package builder uses this endpoint to get customizations in one or multiple categories. This endpoint is only available to administrators.

Request Arguments

Name Type Description Required
elementsToFetch array List of categories to extract customizations.
Example: ['miscellaneous', 'dashboards'] 		True

Response

    {
      "miscellaneous": [
          {
            "map": {
              "files": [],
              "db": []
            },
            "data": {
              "category": "Quick Create Bar",
              "description": "Quick Create Bar from Sugar Header"
            }
          }
        ]
      }
    }
