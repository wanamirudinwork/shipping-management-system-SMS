Overview

Returns modules for which Smart Guide Template is enabled

Summary

This request will return the modules list for which the template is enabled

Request Arguments

Name Type Description Required
module String Record module True
template_id String The id of the Activity Template True

Request Example

{
    "module": "DRI_Workflow_Task_Templates",
    "template_id": "e00ce644-2ce5-11ed-8ea1-0242ac120006",
}

Response

The response will be an object containing the following fields:
Name Type Description
availableModules Array List of modules for which this Smart Guide template is enabled

Response Example

{
    "availableModules" : [
        "Accounts",
        "Cases"
    ]
}
