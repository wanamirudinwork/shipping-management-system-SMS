Overview

Get Prospect Lists Prospects.

Summary

This endpoint will return a set of records for a prospect_lists_prospects table, filtered by an expression.

Request Arguments

Name Type Description Required
filter String The filter expression. Filter expressions are explained below. False
limit Integer A maximum number of records to return. Default is 20. False
offset Integer The number of records to skip over before records are returned. Default is 0. False
deleted Boolean Boolean to show deleted records in the result set. Default is false. False
order String How to sort the returned records, date_modified=ASC will sort in ascending order, DESC in descending.
Example: order[0][date_modified]=ASC 		False

Filter Expressions

There are three types of filters, which can be used in any combination (joined with AND):

date_modified

This will filter the results since a given modification date.
Example
filter[0][date_modified][$gte]=2024/08/18 14:49:33

prospect_list_id

This will filter the results for a specific prospect list id.
Example
filter[0][prospect_list_id]=223098a0-597f-11ef-bb2f-0242ac130004

related_type

This will filter the results of specific prospect related types. The list is comma separated module names.
Example
filter[0][related_type][$in]=Contacts,Leads,Prospects

Response Arguments

Name Type Description
results Array An array of results containing matched records.
