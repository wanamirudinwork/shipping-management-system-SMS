Overview

Make curl request according to webhook bean attributes using custom Request class

Summary

This request will Make curl request according to webhook bean attributes using custom Request class

Request Arguments

Name Type Description Required
module String Record module True
record Object Object containing id of the current record True

Request Example

{
    "module": "CJ_WebHooks",
    "record": {
        "id": "e00ce644-2ce5-11ed-8ea1-0242ac120006"
    },
}

Response

The response will be an object containing the following fields:
Name Type Description
response / N/A String / Void PHP curl response / Return nothing

Response Example

{
    "response" : '{"tries": "3", "url": "https://www.php.net"}'
}
