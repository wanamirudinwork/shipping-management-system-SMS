Overview

Returns the activities for which RSA will be performed

Summary

This request will returns the activities for which RSA will be performed

Request Arguments

Name Type Description Required
parent_type string Module name True
parent_id string Record id True

Request Example

{
    "parent_type" => "Accounts",
    "parent_id" => "36d5a0be-1fad-11ee-b740-462d6d03d2d4"
}

Response

The response will be an array containing the activities e.g.
Name Type Description
Activities Array List of activities to perform RSA on
