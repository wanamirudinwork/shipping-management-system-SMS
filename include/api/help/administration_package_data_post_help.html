Overview

[ADMIN] Retrieve db data for customizations

Summary

Package builder uses this endpoint to get db data for customizations. This endpoint is only available to administrators.

Request Arguments

Name Type Description Required
dashboards array List of dashboard customizations. False
advanced_workflows array List of advanced workflow customizations. False

Response

Db data for customizations
