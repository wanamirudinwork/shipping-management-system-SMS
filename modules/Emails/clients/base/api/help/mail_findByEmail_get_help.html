Overview

Find email object by email address.

Request Arguments

Email
email string Email address to search for

Request Example

{InstanceURL}/rest/v11_25/Mail/findByEmail?email=test123@test.com

    "email_address": "test123@test.com",
    "email_address_caps": "TEST123@TEST.COM",
    "invalid_email": 0,

    "date_modified": "2024-07-23 14:42:14",
