Overview

Reset User Preferences

Summary

This endpoint is used to reset User preferences.

Request Arguments

Name Type Description Required
record String The ID of the user who's preferences are being reset. True
skipResponse Boolean Skip to return any response from the API when set to true. False

Request

{InstanceURL}/rest/v11_23/Users/ffb08d28-7a5c-11ee-a353-02f56818a419/resetPreferences

Optional:
{
    skipResponse: true
}

Response Arguments

Name Type Description
preferences string User ID for which preferences were reset.
