Overview

Fetch template details

Summary

This endpoint will return the details about a template

Request Arguments

Name Type Description
template Array Templates list

    {
        "template":{
           "id":"b5eaad38-4a82-4fbe-8006-df0b88849a69",
           "value":"Project Agreement",
           "name":"Project Agreement",
           "_module":"DocuSignEnvelopes"
        }
     }
