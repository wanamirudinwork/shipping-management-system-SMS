Overview

An API to get data for saved reports.

Summary

This endpoint will return saved reports data of targeted report.

Request Arguments

Name Type Description Required
record String Report Id True

Request Example

GET /rest/v11.17/Reports/activeSavedReport/:id

Response Arguments

Name Type Description
Saved Reports Array Saved Reports def.
