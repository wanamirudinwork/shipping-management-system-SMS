Overview

An API to get filters definitions for a saved report.

Summary

This endpoint will return all the filters defitions for a saved report.

Request Arguments

None.

Request Example

GET /rest/v10/Reports/:id/filter

Response Arguments

Name Type Description
reportDef Array Report def for the chart.
runtimeOperators Array All the operators available for reports.
users Array All the users available for reports.
